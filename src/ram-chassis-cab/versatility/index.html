УНИВЕРСАЛЬНОСТЬ RAM CHASSIS CAB 2021

Сила универсальности

Предустановки для тяжелой работы

Механизм отбора мощности

Обе АКПП Aisin доступны в эксклюзивном исполнении. Доступны с механизмом отбора мощности (МОМ) с левой или правой стороны с конфигурациями 4x4 или 4x2 для обеспечения различных дополнительных возможностей. Кроме того, доступен эксклюзивный для своего класса двигатель Cummins® Turbo Diesel I6 объемом 6,7 л. который позволяет выбрать одну из двух скоростей, включив режим Split-Shaft.

Двойной генератор 440A

Ram chassis cab 2021 оснащен усиленной электропроводкой и спроектирован для генерации электричества повышенной мощности благодаря системе с двумя генераторами переменного тока с лучшей в своем классе максимальной выходной мощностью 440 ампер.

Шесть дополнительных переключателей

В салоне Ram Chassis Cab 2021 года имеется встроенный блок переключателей с шестью вспомогательными кнопками каждая из которых может быть назначена для исполнения своей функции. На моделях, оборудованных механизмом отбора мощности (МОМ), шестой вспомогательный переключатель становится специальным переключателем МОМ.

Сдвоенные топливные баки

Расширьте диапазон обслуживания И географию своих клиентов. Выберите один из самых больших в классе 197-литровых одиночных топливных баков или самый большой в своем классе двойной топливный бак емкостью 280 литров - комбинация, которая состоит из 197-литрового бака и 83-литрового топливного бака. Система включает в себя две независимые топливные горловины. Предусмотрена автоматическая перекачка топлива между баками, избавляя вас от необходимости переключаться между источниками топлива.

Варианты надстроек

Ram chassis cab 2021 может быть сконфигурирован с двумя типами кабин и девятью вариантами колесных баз, предлагая множество вариантов модернизации и возможностей.

Плоская рама для вашего удобства

Удобная для монтажа автомобильная рама оснащена прямыми плоскими направляющими рамы за кабиной, доступна со стандартной длиной оси кабины (CA) и поставляется с предварительно просверленными монтажными отверстиями рамы для различных вариантов крепежа надстроек.

Современный модуль интерфейса автомобиля

Ram chassis cab 2021 оснащен интерфейсным модулем системы транспортного средства (VSIM), который предлагает пользователям удобный доступ к более чем 70 входам/выходам автомобиля, включая средства управления освещением и отбором мощности.
